# user stories
User Registration
As a new user, I want to create an account so that I can place orders and track my purchases.
Priority: High

User Login
As a registered user, I want to log in so that I can securely access my account.
Priority: High

Product Search
As a shopper, I want to search for products by name or category so that I can find items quickly.
Priority: High

Product Filtering
As a shopper, I want to filter products by price, brand, and ratings so that I can narrow down my choices.
Priority: Medium

View Product Details
As a shopper, I want to view detailed product information so that I can make an informed purchase decision.
Priority: High

Add to Cart
As a shopper, I want to add products to my cart so that I can purchase multiple items together.
Priority: High

Checkout
As a shopper, I want to complete checkout securely so that I can place my order successfully.
Priority: High

Order Tracking
As a customer, I want to track my order status so that I know when my order will be delivered.
Priority: Medium

Payment Options
As a customer, I want to choose from multiple payment options so that I can pay conveniently.
Priority: High

Return and Refund
As a customer, I want to request returns and refunds so that I can resolve issues with my orders.
Priority: Medium
